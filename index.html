💡 Cara Memasang di WordPress (Tanpa Plugin):

  1. Buka halaman edit WordPress (Gutenberg Editor atau Classic Editor).
  2. Jika menggunakan Gutenberg (Blok): Klik tombol + (Tambah Blok), cari dan pilih blok HTML Khusus (Custom HTML).
  3. Jika menggunakan Classic Editor: Pindah dari mode visual ke mode Teks (HTML).
  4. Tempel (Paste) kode yang disalin di atas ke dalam blok/area teks tersebut.
  5. Klik Simpan Draf atau Perbarui, lalu lihat halaman tersebut. Selesai!
